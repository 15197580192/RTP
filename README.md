# RSP-query
## build
```
bash build.sh
# all executable program will be generated in the folder ./build/
```
## query config
all query examples of ldbc-snb benchmark are stored in the folder ./queries,please look into the folder and change parameters of query if nessasary.
## run
```
bash run.sh is_path
# eg. bash run.sh ./build/is1
```
